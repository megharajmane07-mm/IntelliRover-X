# IntelliRover X 🚀

An ESP32-based intelligent navigation rover designed for smart movement, obstacle detection, and embedded automation.

## 🔧 Components Used
- ESP32
- L298N Motor Driver
- Ultrasonic Sensor
- 4WD Robot Chassis
- Battery Module

## 💻 Technologies Used
- Arduino IDE
- Embedded C/C++
- IoT Concepts
- PWM Motor Control

## ✨ Features
- Real-time obstacle detection
- Intelligent navigation logic
- Motor speed and direction control
- Sensor integration using ESP32

## 📚 Skills Gained
- Embedded Systems
- Robotics
- Automation
- Problem Solving
- Hardware-Software Integration
